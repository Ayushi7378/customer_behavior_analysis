# Customer Shopping Behavior Analysis

## Overview
This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI.  
The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis across multiple databases, dashboard creation, and report generation.

The goal of the project is to extract meaningful business insights from customer purchase data and present them through interactive dashboards and reports.

---

## Dataset
The dataset contains customer shopping information such as:
- Customer demographics
- Purchase history
- Product categories
- Discounts and promo usage
- Payment methods
- Shopping frequency

Dataset format:
- CSV file

Handled:
- Missing values

- PostgreSQL

- Customer segmentation

Built interactive Power BI dashboards including:
- Sales Overview

- Created analytical report
